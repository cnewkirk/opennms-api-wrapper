# Security Policy

## Supported versions

Only the latest release receives security fixes.

| Version | Supported |
|---------|-----------|
| 0.4.x   | yes       |
| < 0.4   | no        |

## Reporting a vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Report them privately via
[GitHub Security Advisories](https://github.com/cnewkirk/opennms-api-wrapper/security/advisories/new).

You can expect an acknowledgement within 48 hours and a patch or mitigation
plan within 14 days for confirmed issues.  If a CVE is warranted, one will
be requested through GitHub's process.
